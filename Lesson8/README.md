# Практическая работа №8: Работа с картами и геолокацией

## Тема
Интеграция картографических сервисов, работа с геолокацией и построение маршрутов.

## Обзор
Восьмая практическая работа посвящена использованию геопространственных данных и сервисов. В ней рассматривается интеграция различных картографических платформ, таких как OpenStreetMap и Яндекс.Карты.

## Содержание модулей

| Модуль | Описание | Ключевые концепции |
| :--- | :--- | :--- |
| **osmmaps** | Пример использования карт на основе OpenStreetMap. Демонстрирует отображение карты, добавление маркеров и, возможно, базовое управление камерой карты. | OpenStreetMap, библиотека `osmdroid`, отображение тайлов, работа с координатами. |
| **yandexdriver** | Приложение, связанное с использованием сервиса Яндекс.Такси. Вероятно, демонстрирует работу с геолокацией, отображение текущего местоположения и построение маршрутов. | Геолокация (`LocationManager` или `FusedLocationProviderClient`), запросы разрешений, отображение местоположения на карте. |
| **yandexmaps** | Пример интеграции и использования Яндекс.Карт. Показывает, как подключить SDK Яндекс.Карт, отобразить карту, добавить интерактивные элементы (маркеры, полилинии). | SDK Яндекс.Карт, получение API-ключа, управление картой, добавление объектов на карту. |

## Ключевые навыки, освоенные в работе
1.  **Интеграция карт:** Подключение и настройка различных картографических сервисов.
2.  **Геолокация:** Получение текущего местоположения пользователя и отслеживание его изменений.
3.  **Отображение данных:** Нанесение маркеров, полилиний и других геообъектов на карту.

## Технические детали
*   **Язык программирования:** Java/Kotlin.
*   **Среда разработки:** Android Studio.
*   **Целевая платформа:** Android SDK.
