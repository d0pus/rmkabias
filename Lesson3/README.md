# Практическая работа №3: Activity, Intents, Fragments и обмен данными

## Тема
Углубленное изучение компонентов `Activity` и `Fragment`, а также механизмов обмена данными между ними и другими приложениями с использованием `Intent`.

## Обзор
Третья практическая работа фокусируется на ключевых строительных блоках Android-приложений: Activity и Fragment, а также на том, как они взаимодействуют друг с другом и с внешним миром через систему Intent.

## Содержание модулей

| Модуль | Описание | Ключевые концепции |
| :--- | :--- | :--- |
| **favoritebook** | Демонстрация передачи сложных объектов или большого объема данных между Activity. Вероятно, используется сериализация (`Serializable` или `Parcelable`) для передачи объекта "Книга". | Передача объектов между Activity, сериализация данных, использование `startActivityForResult` для получения результата. |
| **intentapp** | Примеры явных (`Explicit`) и неявных (`Implicit`) Intent. Явные Intent используются для запуска конкретного компонента, неявные — для запроса выполнения действия системой. | Явные и неявные Intent, действия (`ACTION_VIEW`, `ACTION_SEND`), категории, передача дополнительных данных (`Extras`). |
| **sharer** | Приложение, демонстрирующее механизм "Поделиться" (Share). Использует неявный Intent с действием `ACTION_SEND` для отправки контента (текста, изображений) в другие приложения. | `ACTION_SEND` Intent, MIME-типы данных, предоставление доступа к файлам через `FileProvider`. |
| **simplefragmentapp** | Базовый пример использования `Fragment`. Показывает, как встроить фрагмент в Activity, управлять его жизненным циклом и взаимодействовать с ним. | Жизненный цикл `Fragment`, `FragmentManager`, статическое и динамическое добавление фрагментов, взаимодействие `Activity-Fragment`. |
| **systemintentsapp** | Примеры использования стандартных системных Intent для выполнения общих задач, таких как вызов номера, открытие веб-страницы, отправка SMS. | Использование системных действий (`ACTION_DIAL`, `ACTION_WEB_SEARCH`), проверка наличия приложения-обработчика. |

## Ключевые навыки, освоенные в работе
1.  **Управление фрагментами:** Эффективное использование фрагментов для создания гибких и модульных интерфейсов, поддерживающих различные размеры экрана.
2.  **Продвинутое использование Intent:** Различение и применение явных и неявных намерений для внутренней и внешней коммуникации.
3.  **Обмен данными:** Реализация механизмов передачи данных между компонентами приложения и между приложениями.
