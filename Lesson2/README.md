# Практическая работа №2: Жизненный цикл Activity и взаимодействие с пользователем

## Тема
Изучение жизненного цикла компонента `Activity`, создание диалоговых окон, уведомлений и различных форм взаимодействия с пользователем.

## Обзор
В данной практической работе детально рассматривается, как управляется состояние главного компонента Android-приложения — `Activity`. Особое внимание уделяется методам обратного вызова жизненного цикла, а также механизмам оповещения пользователя о событиях в приложении.

## Содержание модулей

| Модуль | Описание | Ключевые концепции |
| :--- | :--- | :--- |
| **activitylifecycle** | Демонстрация работы методов жизненного цикла `Activity` (`onCreate()`, `onStart()`, `onResume()`, `onPause()`, `onStop()`, `onDestroy()`). Позволяет отслеживать смену состояний при различных действиях пользователя и системы. | Жизненный цикл `Activity`, логирование состояний, сохранение и восстановление состояния (`onSaveInstanceState()`). |
| **dialog** | Примеры использования различных типов диалоговых окон для взаимодействия с пользователем. Включает стандартные `AlertDialog`, а также, возможно, кастомные диалоги. | `AlertDialog`, обработка выбора пользователя в диалоге, использование `DialogFragment`. |
| **interfilter** | Демонстрация использования фильтров намерений (Intent Filters) для обработки неявных запросов от других приложений или системы. Позволяет приложению реагировать на определенные типы данных или действий. | `Intent Filter`, неявные намерения, обработка данных, передаваемых через Intent. |
| **multiactivity** | Пример приложения с несколькими `Activity` и навигацией между ними. Показывает, как запускать новые экраны и передавать данные между ними. | Запуск новой `Activity`, передача данных через `Intent` (`putExtra`), стек Activity. |
| **notificationapp** | Создание и управление системными уведомлениями. Демонстрирует, как оповещать пользователя о событиях, когда приложение неактивно. | `NotificationManager`, создание канала уведомлений (для Android 8.0+), настройка внешнего вида уведомления. |
| **toastapp** | Использование всплывающих сообщений (Toast) для краткого информирования пользователя без прерывания его работы. | `Toast` сообщения, настройка длительности отображения. |

## Ключевые навыки, освоенные в работе
1.  **Управление состоянием Activity:** Понимание и правильное использование методов жизненного цикла для управления ресурсами и данными.
2.  **Информирование пользователя:** Применение `Toast` и системных уведомлений для эффективной коммуникации с пользователем.
3.  **Межкомпонентное взаимодействие:** Использование `Intent` для запуска других компонентов и `Intent Filter` для приема внешних запросов.
4.  **Диалоговые окна:** Создание модальных окон для запроса информации или подтверждения действий.

*   **Среда разработки:** Android Studio.
