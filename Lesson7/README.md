# Практическая работа №7

## Тема
Сетевое взаимодействие и сторонние библиотеки.

## Содержание
В данной директории представлены следующие модули/проекты, демонстрирующие работу с сетевыми возможностями и интеграцию сторонних сервисов:
- **firebaseauth**: Пример использования сторонней библиотеки для аутентификации через Firebase.
- **httpurlconnection**: Приложение для выполнения HTTP-запросов и получения данных (например, IP-адреса и информации о погоде).
- **timeservice**: Пример сетевого взаимодействия с использованием сокетов для получения времени с сервера.

*Примечание: В папке также содержатся скриншоты (`.png` файлы), иллюстрирующие процесс выполнения практической работы.*
