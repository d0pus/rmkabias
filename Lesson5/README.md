# Практическая работа №5

## Тема
Основы использования аппаратных возможностей мобильных устройств.

## Содержание
В данной директории представлены следующие модули/проекты, демонстрирующие работу с аппаратными возможностями устройства:
- **accelerometer**: Пример работы с датчиками, в частности, с акселерометром.
- **audiorecord**: Приложение для записи и воспроизведения аудио с использованием микрофона.
- **camera**: Приложение для работы с камерой устройства.

*Примечание: В папке также содержатся скриншоты (`.png` файлы), иллюстрирующие процесс выполнения практической работы.*
