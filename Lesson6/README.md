# Практическая работа №6: Хранение данных

## Тема
Изучение различных способов постоянного хранения данных в Android-приложениях, включая базы данных, внутреннее и внешнее хранилище, а также защищенные настройки.

## Обзор
Шестая практическая работа охватывает фундаментальные аспекты работы с данными: от простых пар "ключ-значение" до структурированных реляционных баз данных.

## Содержание модулей

| Модуль | Описание | Ключевые концепции |
| :--- | :--- | :--- |
| **employeedb** | Пример работы с реляционными базами данных с использованием SQLite. Демонстрирует создание базы данных, таблиц, выполнение CRUD-операций (Create, Read, Update, Delete). | SQLite, `SQLiteOpenHelper`, `ContentValues`, курсоры, основы реляционных баз данных. |
| **internalfilestorage** | Демонстрация хранения данных во внутренней памяти устройства. Файлы, сохраненные таким образом, доступны только самому приложению. | Внутреннее хранилище, `openFileOutput()`, `openFileInput()`, чтение и запись текстовых данных. |
| **notebook** | Приложение-записная книжка, использующее хранение данных. Вероятно, демонстрирует сохранение заметок в файл или базу данных. | Интеграция UI с механизмом хранения, сохранение и загрузка пользовательского контента. |
| **securesharedpreferences** | Пример использования `SharedPreferences` с дополнительной защитой. Показывает, как хранить небольшие объемы конфиденциальных данных в зашифрованном виде. | `SharedPreferences`, шифрование данных, использование библиотек для безопасного хранения (например, `EncryptedSharedPreferences`). |

## Ключевые навыки, освоенные в работе
1.  **Постоянное хранение:** Выбор подходящего механизма хранения данных в зависимости от их типа и объема (файлы, настройки, БД).
2.  **Работа с SQLite:** Создание и управление локальной базой данных.
3.  **Безопасность данных:** Применение методов для защиты конфиденциальных данных, хранящихся на устройстве.
