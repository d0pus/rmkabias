# Практическая работа №5: Основы использования аппаратных возможностей мобильных устройств

## Тема
Интеграция и работа с аппаратными возможностями мобильных устройств, включая датчики, камеру и микрофон.

## Обзор
Пятая практическая работа посвящена использованию встроенных аппаратных компонентов смартфона. Это включает доступ к данным с датчиков (акселерометр), захват изображений с камеры и запись аудио с микрофона.

## Содержание модулей

| Модуль | Описание | Ключевые концепции |
| :--- | :--- | :--- |
| **accelerometer** | Приложение для работы с датчиками, в частности, с акселерометром. Демонстрирует, как получить данные о движении устройства и отобразить их в реальном времени. | `SensorManager`, `Sensor`, `SensorEventListener`, получение данных с датчиков, обработка изменений. |
| **audiorecord** | Приложение для записи и воспроизведения аудио с использованием микрофона. Показывает, как запросить разрешение на использование микрофона и управлять процессом записи/воспроизведения. | Разрешения времени выполнения (`RECORD_AUDIO`), `MediaRecorder` для записи, `MediaPlayer` для воспроизведения, работа с файловой системой. |
| **camera** | Приложение для работы с камерой устройства. Демонстрирует, как запустить системное приложение камеры через Intent и получить сделанный снимок. | Разрешения времени выполнения (`CAMERA`), `Intent` для запуска камеры, `FileProvider` для безопасного доступа к файлам, обработка возвращенного изображения. |

## Ключевые навыки, освоенные в работе
1.  **Работа с датчиками:** Инициализация и чтение данных с аппаратных датчиков устройства.
2.  **Управление мультимедиа:** Запись аудио и взаимодействие с камерой для захвата изображений.
3.  **Система разрешений:** Запрос и обработка разрешений времени выполнения (Runtime Permissions) для доступа к чувствительным аппаратным ресурсам.

## Технические детали
*   **Язык программирования:** Java/Kotlin.
*   **Среда разработки:** Android Studio.
*   **Целевая платформа:** Android SDK.
*   **Дополнительно:** В папке содержатся скриншоты (`.png` файлы), иллюстрирующие процесс выполнения практической работы.
