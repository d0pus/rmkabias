# Практическая работа №4

## Тема
Многопоточность, фоновые задачи и службы.

## Содержание
В данной директории представлены следующие модули/проекты, демонстрирующие работу с многопоточностью и фоновыми задачами:
- **cryptoloader**: Пример использования `Loader` для выполнения асинхронных операций.
- **data_thread**: Демонстрация работы с потоками данных.
- **looper**: Пример использования `Looper` и `Handler` для организации очереди сообщений.
- **serviceapp**: Примеры использования `Service` для выполнения длительных операций в фоне.
- **thread**: Базовый пример работы с классом `Thread`.
- **workmanager**: Пример использования `WorkManager` для отложенного выполнения задач.
