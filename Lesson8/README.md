# Практическая работа №8

## Тема
Работа с картами и геолокацией.

## Содержание
В данной директории представлены следующие модули/проекты, демонстрирующие интеграцию картографических сервисов и работу с геолокацией:
- **osmmaps**: Пример использования карт на основе OpenStreetMap.
- **yandexdriver**: Приложение, связанное с использованием сервиса Яндекс.Такси (вероятно, для демонстрации работы с геолокацией и маршрутами).
- **yandexmaps**: Пример интеграции и использования Яндекс.Карт.
