# Практическая работа №6

## Тема
Хранение данных.

## Содержание
В данной директории представлены следующие модули/проекты, демонстрирующие различные способы хранения данных в Android-приложениях:
- **employeedb**: Пример работы с базами данных SQLite.
- **internalfilestorage**: Демонстрация хранения данных во внутренней памяти устройства.
- **notebook**: Приложение-записная книжка, использующее хранение данных.
- **securesharedpreferences**: Пример использования `SharedPreferences` с дополнительной защитой.

*Примечание: В папке также содержатся скриншоты (`.png` файлы), иллюстрирующие процесс выполнения практической работы.*
