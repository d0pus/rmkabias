# Практическая работа №3

## Тема
Жизненный цикл Activity, Intents, Fragments и обмен данными.

## Содержание
В данной директории представлены следующие модули/проекты:
- **favoritebook**: Пример использования Activity и передачи данных.
- **intentapp**: Примеры явных и неявных Intent.
- **sharer**: Приложение для обмена данными с другими приложениями.
- **simplefragmentapp**: Пример использования Fragments.
- **systemintentsapp**: Примеры использования системных Intent.
